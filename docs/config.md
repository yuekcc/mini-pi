# 配置目录

默认在 `~/.config/mp/`，在启动通过 `--config <dir_path>` 或 `-c <dir_path>` 可以指定项目目录。

**内容**

```sh
~/.config/mp/
    log/    # 日志
    temp/   # 临时文件
    agents/     # 自定义 agent
        my-agent.md
        ...
    commands/   # 自定义 / 命令
        my-slash-command.md
        ...
    config.json # 配置
```
